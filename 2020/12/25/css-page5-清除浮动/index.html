清除浮动

浮动的介绍

  1. 作用：引入 float 属性是为了能让 web 开发人员实现简单的布局，包括在一列文本中浮动的图像，文字环绕在它的左边或右边。
  2. 原理：float 元素脱离了 normal flow 就相当于进入了一个平行空间，不再与其后边及父级块级元素发生反应，这一特性使得 float 元素会导致很多副作用
  3. 副作用:float 元素会导致父元素坍塌，垂直外边距重叠等许多副作用

清除浮动的方法

使用邻接元素处理，为 float 元素后的元素添加 clear 属性

  • 添加clear后，清除两侧浮动

利用:after伪元素

  • 添加伪类，清除浮动。实现元素末尾添加一个看不见的块元素（Block element）清理浮动

添加父容器，触发父容器的BFC

  • 修改父元素的 owerflow 属性（auto或hidden），触发BFC，清除浮动
