Vue学习笔记7——编写天气预报应用

天气预报的功能

1. 回车查询：按下回车可以查询相应的城市天气 2. 点击查询：点击下方的城市名字可以查询相应城市的天气

回车查询

逻辑

  1. 按下回车：v-on绑定事件 @keyup.enter
  2. 查询数据：axios请求接口的数据 用v-model获取用户的输入
  3. 渲染数据：v-for绑定请求数据（数组形式） that保存this的数据

点击查询

逻辑

  1. 点击城市：v-on绑定事件 @click
  2. 查询数据
  3. 渲染数据

代码如下，由于篇幅原因，CSS样式代码省略

<div class="wrap" id="app">
    <div class="search_form">
        <div class="txt">天气预报</div>
        <div class="form_group">
            <input type="text" v-model="city" @keyup.enter="searchWeather" class="input_txt" placeholder="请输入查询的天气"/>
            <button class="input_sub">搜 索</button>
        </div>
        <div class="hotkey">
            <a href="javascript:;" @click="changeCity('北京')">北京</a>
            <a href="javascript:;" @click="changeCity('上海')">上海</a>
            <a href="javascript:;" @click="changeCity('广州')">广州</a>
            <a href="javascript:;" @click="changeCity('深圳')">深圳</a>
        </div>
    </div>
    <ul class="weather_list">
        <li v-for="item in weatherList">
            <div class="info_type">
                <span class="iconfont">{{item.type}}</span>
                <div class="info_temp">
                    <b>{{item.low}}</b>
                    <b>{{item.high}}</b>
                </div>
                <div class="info_date"><span>{{item.date}}</span></div>
            </div>
        </li>
    </ul>
</div>
var app=new Vue({
    el:"#app",
    data:{
        city:'',
        //存放天气的数组
        weatherList:[]
    },
    methods:{
        searchWeather:function(){
            // console.log('天气查询');
            // console.log(this.city);
            //调用接口
            //保存this
            var that=this;
            axios.get('http://wthrcdn.etouch.cn/weather_mini?city='+this.city)
            .then(function(response){
                // console.log(response);
                // console.log(response.data.data.forecast);
                that.weatherList=response.data.data.forecast;
            })
            .catch(function(err){})
        },
        changeCity:function(city){
            this.city=city;
            // console.log(city);
            this.searchWeather();
        }
    }
})
