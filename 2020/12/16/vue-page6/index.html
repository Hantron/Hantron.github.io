Vue学习笔记6——axios

axios

  1. 功能强大的网络请求库，基于ajax
  2. 可结合axios+vue进行网络数据开发应用
  3. 使用get或post方法发送对应的请求
  4. then方法中的回调函数在请求成功或失败时触发
  5. 通过回调函数的形参可以获取响应内容或错误信息

axios的使用

<!-- axios在线地址 -->
<script src="http://unpkg.com/axios/dist/axios.min.js"></script>
<!-- axios的使用，get请求，其中key为请求参数 -->
axios.get(地址?key=value&key2=value2).then(function(response){},function(err){})
<!-- axios的使用，post请求，其中key为请求参数 -->
axios.get(地址,{key=value,key2=value2}).then(function(response){},function(err){})

关于接口

接口的属性有请求地址，请求方法，请求参数等

<input type="button" value="get请求" class="get">
<input type="button" value="post请求" class="post">

补充：querySelector() 方法返回文档中匹配指定 CSS 选择器的第一个元素。

/*
    接口1:随机笑话
    请求地址:http://autumnfish.cn/api/joke/list1234?num=3
    请求方法:get
    请求参数:num(笑话条数，数字)
    响应内容:随机笑话
*/
document.querySelector(".get").onclick=function(){
    axios.get("http://autumnfish.cn/api/joke/list1234?num=3")
    .then(function(response){
        console.log(response);
    },function(err){
        console.log(err);
    })
}

/*
    接口2:用户注册
    请求地址:http://autumnfish.cn/api/user/reg
    请求方法:post
    请求参数:username(用户名，字符串)
    响应内容:注册成功或失败
*/
document.querySelector(".post").onclick=function(){
    axios.post("http://autumnfish.cn/api/user/reg",{username:"jack"})
    .then(function(response){
        console.log(response);
    },function(err){
        console.log(err);
    }) 
}

axios+vue

<div id="app">
    <input type="button" value="获取笑话" @click="getjoke">
    <p>{{joke}}</p>
</div>
/*
    接口：随机获取一条笑话
    请求地址:https://autumnfish.cn/api/joke
    请求方法:get
    请求参数:无
    响应内容：随即笑话
*/
var app=new Vue({
    el:"#app",
    data:{
        joke:"很好笑的笑话"
    },
    methods:{
        getjoke:function(){
            var that=this;
            axios.get("https://autumnfish.cn/api/joke").then
            (function(response){
                console.log(response.data);
                that.joke=response.data;
            },function(err){})
        }
    },
})
* 注意，上述代码中，axios回调函数的this对象已经改变了，无法访问到请求到的data中的数据 * 所以，在then方法外部，用that保存this对象，即可在回调函数中直接使用保存的this即可
