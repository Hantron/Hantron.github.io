Vue程序的编写

声明式渲染

采用简洁的模板语法来声明式地将数据渲染进 DOM 的系统，例子如下：

<div id="app">
    {{message}}
</div>
var app=new Vue({
    el:"#app",
    data:{
        message:"Hello Vue!"
    }
})
// 输出
Hello Vue!
总结
  1. 创建Vue实例对象，设置el属性和data属性
  2. 采用简洁的模板语法来渲染数据到页面上
那el属性和data属性如何应用呢
el属性:挂载点
  1. 作用:用来设置Vue实例挂载(管理)的元素
  2. 作用范围：管理命中的元素及其后代元素
  3. 选择器：建议ID选择器，也可以使用标签来选择，但不能使用HTML和BODY标签
    data属性:
  • data中可以写复杂类型的数据,如数组，对象等。
  • 例子如下
<div id="app">
    {{message}}
    <h2>{{school.name}}{{school.mobile}}</h2>
    <ul>
        <li>{{college[0]}}</li>
    </ul>
</div>
var app=new Vue({
    el:"#app",
    data:{
        message:"Hello Vue!",
        school:{
            name:"EKKO",
            mobile:"88888888"
        },
        college:["大数据","中德","交通","新能源"]
    }
})
// 输出
Hello Vue!
EKKO88888888
大数据
