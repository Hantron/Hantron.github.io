CSS学习笔记-position属性

发表于2021-07-11|更新于2021-07-12|CSS
|阅读量:

position的定义

  • 用来规定元素的定位类型
  • 常见的值有static、relative、absolute、fixed、inherit、sticky

position的类型

  • static
    • 默认值，元素正常出现在文档流中
  • relative相对定位
    • 相对于该元素在文档流中的正常位置进行偏移定位，原占位仍保留(即不脱离文档流)。
  • absolute绝对定位
    • 相对于直接父级元素进行偏移定位，脱离了文档流
    • absolute一般是配合relative进行更灵活的定位，就是absolute基于relative定位的框架层，然后基于该层最左上点作为基点进行偏移定位
  • fixed固定定位
    • 相对于浏览器窗口进行定位，即固定定位是以浏览器窗口的最左上点作为基点进行偏移定位的。
  • 以上三种定位，被定位的元素的位置都是通过 “left”, “top”, “right” 以及 “bottom” 这四个属性进行指定的。
  • 同时，以上三种定位可以通过设置z-index设置z轴的层级关系

新属性sticky

  • sticky是在目标区域内随着页面的滚动而保持在固定位置，超出这个区域后就就随着目标区域滚动。

  • 结合了 position:relative 和 position:fixed 两种定位功能于一体的特殊定位，适用于一些特殊场景。

  • sticky的使用限制：要设置top、bottom、left、right其中之一才能使sticky生效。

  • sticky的特点：

    • 不会触发BFC
    • z-index无效
    • sticky是容器相关的，sticky的特性只会在其容器内生效
