State

什么是State

  • State是组件的内部属性，可以根据用户的交互改变，但是不能用来作组件间传递数据。
  • State是局部或者是封装的，除了拥有并设置了它的组件，其它组件都无法访问
  • 构造函数中，可以通过this.state初始化state。但是修改state时，应该用setState()更新
