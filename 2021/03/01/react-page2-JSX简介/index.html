React笔记2——JSX简介

什么是JSX

  • 这个标签语法被称为JSX，是JS的语法拓展，一般会在React中配合使用JSX,JSX可以很好地描述UI呈现的交互形式。
  • JSX具有JS的全部功能，并且可以生成React元素

为什么要用JSX

  • React认为渲染逻辑本质上与其他UI逻辑内在耦合。因此在代码中将JSX和UI放在一起时，会在视觉上有辅助作用。

JSX的简单实用

  • 声明一个name变量，然后在JSX使用它，并将它包在大括号中

JSX也是表达式

  • 编译之后，JSX 表达式会被转为普通 JavaScript 函数调用，并且对其取值后得到 JavaScript 对象
  • 在 if 语句和 for 循环的代码块中使用 JSX，将 JSX 赋值给变量，把 JSX 当作参数传入，以及从函数中返回 JSX

JSX特定属性

  • 可以使用引号，将属性值指定为字符串字面量

  • 可以使用大括号，在属性值中插入一个JS表达式

使用 JSX 指定子元素

  • 一个标签里面没有额呢绒，可以用’/>’来闭合标签

JSX表示对象

  • Babel会把JSX转译成一个React.createElement()函数调用
  • React.createElement()会预先执行一些检查，实际上它创建了一个对象
  • 这些对象被称为React元素，用于描述屏幕的内容。React通过读取这些对象，然后用它们构建DOM和保持更新
